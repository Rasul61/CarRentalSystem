# CarRentalSystem
CarRentalSystem is a web-based application that allows users to register, manage cars, and create rental orders. The system provides a clean and structured backend architecture built with modern Java technologies.

## Features
- User Registration and Authentication
- Viewing List of Available Cars
- *Creating and Managing Cars* (add, update, delete)
- Creating Rental Orders
- API Documentation via Swagger UI
- Database Versioning with Liquibase

### Technologies Used
- *Java 21*
- *Spring Boot*
- *Spring Data JPA*
- *PostgreSQL*
- *Liquibase*
- *Docker / Docker Compose*
- *Swagger / OpenAPI*
